Code for "python-fill" for developers:

Iconify SVG framework makes using icons as easy as icon fonts. To use "python-fill" in HTML, add this code to the document:

<span class="iconify" data-icon="akar-icons:python-fill"></span>

Iconify SVG framework will load icon data from Iconify API and replace that placeholder with SVG.

Make sure you import Iconify SVG framework:

<script src="https://code.iconify.design/2/2.2.1/iconify.min.js"></script>

Click here for more information about Iconify SVG framework.

Iconify tries to unify all icon sets. You can use the same code no matter what icon set you choose. You can mix icons from multiple icon sets on the same page.
